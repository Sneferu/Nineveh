Starting a tmux session and window group:
	tmux new -s <name>

Starting a tmux session on an old window group
	tmux new-session -t <old-name>
	tmux new -s <new-name> -t <old-name>

To check available sessions
	tmux ls

To reenter a session
	tmux attach-session -t <number|name>
	tmux a -t <number|name>

	OR to enter last created session

	tmux a #

Inputting a tmux command
	ctrl+b <command>

Entering tmux prompt ($>)
	ctrl+b :

To detach a session:
	ctrl+b d

--------------------------------------------------------------------------

To create a new window
	ctrl+b c

To rename a window
	ctrl+b ,

--------------------------------------------------------------------------

To split a pane horizontally
	ctrl+b "

To split a pane vertically
	ctrl+b %

To move pane to pane:
	ctrl+b <arrow_key>

To kill a pane
	ctrl+b x

To resize a pane
	$> resize-pane -<U|D|R|L> <number_of_lines>
	OR
	Hold ctrl+b, and use arrow keys

To scroll
	ctrl+b [
	Then use arrow keys

Flow:
	First, create new session
		tmux -s base
	Next, create new window for second session
		ctrl+b c
	Finally, attach second session
		tmux new-session -t base
